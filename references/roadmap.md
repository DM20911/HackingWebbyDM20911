# Roadmap dm20911 Offensive Web

- HTTP / HTTPS internals
- Cookies / Sessions
- Burp Suite (proxy, repeater, intruder)
- OWASP Top 10

- ffuf, nuclei

- Mobile ↔ Web correlation

## Nivel 4 — Elite
